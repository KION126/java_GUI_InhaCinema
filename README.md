# 대학교 JavaPrograming과목 기말 프로젝트

### 영화관 예매 윈도우 애플리케이션
* Netbeans GUI로 영화관 결제, 좌석 예매 윈도우 애플리케이션
* Text Captcha API를 적용시켜 사용자가 사람인지 컴퓨터 프로그램인지 판별, 보안강화
* 실시간 영화관 좌석 예매를 위한 DB서버 구축 및 로직 작
